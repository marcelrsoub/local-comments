# Contributing to Local Comments

Thank you for your interest in contributing to Local Comments!

## Getting Started

1. Fork the repository on GitHub

2. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/local-comments.git
   cd local-comments
   ```

3. Install dependencies:
   ```bash
   yarn install
   ```

4. Create a feature branch:
   ```bash
   git checkout -b feature/my-feature
   ```

## Development

See [DEVELOPMENT.md](./DEVELOPMENT.md) for detailed instructions on:

- Setting up your development environment
- Compiling and testing locally
- Project structure overview
- Debugging tips

## Submitting Changes

1. **Make your changes** in your feature branch

2. **Test thoroughly** using the local testing methods described in [DEVELOPMENT.md](./DEVELOPMENT.md)

3. **Commit** with descriptive messages:
   ```bash
   git commit -m "feat: add new feature description"
   ```
   
   Commit message prefixes:
   - `feat:` - New feature
   - `fix:` - Bug fix
   - `docs:` - Documentation changes
   - `refactor:` - Code refactoring
   - `test:` - Adding tests
   - `chore:` - Maintenance tasks

4. **Push** to your fork:
   ```bash
   git push origin feature/my-feature
   ```

5. **Open a Pull Request** on GitHub

## Code Style

- Run `yarn lint` before committing to check for issues
- Follow existing code patterns and conventions
- Use TypeScript types appropriately
- Keep functions focused and readable

## Documentation

- Update `CHANGELOG.md` if your change affects users
- Update `README.md` if you change functionality
- Add inline comments for complex logic

## Reporting Issues

When reporting bugs or requesting features:

1. Check if the issue already exists in [GitHub Issues](https://github.com/marcelrsoub/local-comments/issues)

2. Create a new issue with:
   - Clear, descriptive title
   - Steps to reproduce (for bugs)
   - Expected vs. actual behavior
   - Your VS Code version
   - Your operating system
   - Screenshots if helpful

## Questions?

Feel free to open an issue with your question, and we'll help you out!

Thank you for contributing! 🎉
